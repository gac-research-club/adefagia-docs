## Armor Specification

Armor Specification outlines the protective attributes of robot armor, including:

| Type      | Defense Constant | Health Point |
| --------- | ---------------- | ------------ |
| Aggresive | 2                | 30           |
| Balance   | 1                | 40           |
| Defensive | 0.7              | 50           |

### Description : 

- **Type**: Defines the type of armor, such as Aggressive, Balanced, or Defensive.
- **Defense Constant**: Indicates the constant value of defense provided by the armor.
- **Health Point**: Specifies the maximum health points that the armor provides.

Armor Specification is vital for determining a robot's durability and ability to withstand attacks.
