## Item Specification

Item Specification details the effects and attributes of various in-game items, including:


| Type        | Effect              | Round |
| ----------- | ------------------- | ----- |
| Buff Attack | +2 Damage Constant  | 1     |
| Buff Armor  | +1 Defense Constant | 1     |
| Repair      | +10 Health Point    | 1     |

### Description : 

- **Type**: Specifies the category or type of item, such as Buff Attack, Buff Armor, or Repair.
- **Effect**: Describes the item's effect or bonus, such as increased damage or health point restoration.
- **Round**: Indicates the duration or number of rounds the item's effect lasts.
